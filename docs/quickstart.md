```shell
$ git clone https://github.com/TelenorFrontend/docs-scaffold.git
```
```shell
$ cd docs-scaffold
```

```shell
$ npm install
```

```shell
$ npm run dev
```

Go to [localhost:8888](http://localhost:8888/).
