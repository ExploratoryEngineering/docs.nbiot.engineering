Pure html pages will be wrapped in the layout, but otherewise remain as-is.

Script tags

If the page contains script tags anything that should be run should be set to the `window.pageReady` function like so. This is to make sure the script is properly run on Barba.js page transitions. 
<script>
  window.pageReady = (container) => {
    console.log("The container containing the page is " + container);
  }
</script>
